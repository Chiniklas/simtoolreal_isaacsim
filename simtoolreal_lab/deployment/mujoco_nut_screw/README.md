# MuJoCo Nut-Screw Deployment

This folder mirrors the existing no-ROS MuJoCo deployment, but the replay scene is
now a concrete MJCF file instead of a runtime rewrite of the base iiwa+SHARPA
scene. The scene loads a fixed M12 screw on the table and a movable M12 nut as
the policy object.

The scene expects OBJ meshes generated by:

```bash
python simtoolreal_lab/tasks/sharpa_nutscrew_pick_place_screw/tests/screw_generator.py \
  --families M8 M10 M12 M16 M20 \
  --formats usd obj stl \
  --overwrite
```

Quick scene-only viewer:

```bash
python -m simtoolreal_lab.deployment.mujoco_nut_screw.mujoco_sim
```

The default MJCF lives at:

```text
simtoolreal_lab/deployment/mujoco_nut_screw/scenes/iiwa_sharpa_nut_screw.xml
```

It includes the task-specific bodies from:

```text
simtoolreal_lab/deployment/mujoco_nut_screw/scenes/nut_screw_setup_m12.xml
```

Policy rollout:

```bash
python -m simtoolreal_lab.deployment.mujoco_nut_screw.mujoco_env_nut_screw \
  --family M12 \
  --screw M12X30 \
  --nut M12_nut \
  --press-enter-to-execute
```

By default the nut uses six invisible box collision proxies so the center hole is
not filled by a convex mesh collision. The screw uses invisible cylinder proxies
for the head and shaft. To test different mesh/collision choices, edit the setup
include or pass a different `--scene-xml-path`.
